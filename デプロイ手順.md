# 手相鑑定ツールをネットに公開する手順（GitHubアカウントがある方向け）

パソコンの画面操作だけで、10分程度で完了します。

---

## ステップ1: GitHubにプロジェクトをアップロードする

### 1-1. 新しいリポジトリを作る

1. [github.com](https://github.com) にログイン
2. 右上の **＋** マーク → **New repository** をクリック
3. **Repository name** に `tesou-kantei-tool` と入力
4. **Public** を選ぶ
5. **Create repository** をクリック

### 1-2. ファイルをアップロードする

1. 作ったリポジトリのページで **「uploading an existing file」** のリンクをクリック  
   （または **Add file** → **Upload files**）

2. 次のファイル・フォルダを**ドラッグ＆ドロップ**でアップロード：
   - `index.html`
   - `app.js`
   - `palm-data.js`
   - `style.css`
   - `vercel.json`
   - `api` フォルダ（中身の enhance.js を含む）
   - `images` フォルダ

3. **注意**: `.env` ファイルは**アップロードしないでください**（秘密のキーが入っています）

4. 下の **Commit changes** をクリック

---

## ステップ2: Vercelでデプロイする

### 2-1. Vercelに登録

1. [vercel.com](https://vercel.com) を開く
2. **Continue with GitHub** をクリック
3. GitHubの画面で **Authorize Vercel** を許可

### 2-2. プロジェクトをインポート

1. Vercelの画面で **Add New...** → **Project** をクリック
2. **Import Git Repository** のところで、`tesou-kantei-tool` を選ぶ
3. **Import** をクリック

### 2-3. 環境変数を設定する

1. インポート画面の **Environment Variables** を開く
2. **Key** に `OPENAI_API_KEY` と入力
3. **Value** に、あなたのOpenAIのAPIキーを貼り付け
   - `.env` ファイルを開いて、`=` の右側の長い文字列をコピー
4. **Add** をクリック

### 2-4. デプロイを実行

1. **Deploy** をクリック
2. 1〜2分待つ
3. 完了すると **Visit** というボタンが表示される → クリックすると公開URLが開きます

---

## 完了！

表示されたURLが、あなたの手相鑑定ツールの本番アドレスです。  
このURLを共有すれば、他の人も使えます。

「AIで文章を整える」ボタンも、このURL上で正しく動作します。
